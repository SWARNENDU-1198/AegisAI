    Create a new file `backend/services/categorization_service.py` with:
    1. Mappings of suffixes (lowercase extension keys) to logical categories (`Media`, `Documents`, `Code`, `Archives`, `Temp`).
    2. Magic bytes byte strings mapping to categories:
       - `b"\xff\xd8\xff"`: "Media" (JPEG)
       - `b"\x89PNG\r\n"`: "Media" (PNG)
       - `b"GIF8"`: "Media" (GIF)
       - `b"%PDF"`: "Documents" (PDF)
       - `b"PK\x03\x04"`: "Archives" or "Documents" (depending on whether extension matches docx/xlsx/pptx)
       - `b"MZ"`: "System"
    3. Define `determine_category(file_path: str, extension: str) -> str`:
       - First, check extension lowercased against the suffix mapping. If found, return it immediately.
       - If extension is unrecognized or empty, attempt to read the first 8 bytes of the file. Check if any byte signatures match.
       - If zip container `b"PK\x03\x04"` matches, verify the extension to return "Documents" (for docx, xlsx, pptx) or "Archives".
       - Return "Other" as fallback. Use a try-except block to handle file read permissions or non-existence gracefully.
  </action>

       ```python
       from sqlalchemy import func
       
       def get_storage_by_category():
           db = SessionLocal()
           try:
               results = (
                   db.query(
                       FileRecord.category,
                       func.count(FileRecord.id).label("count"),
                       func.sum(FileRecord.size).label("total_size")
                   )
                   .group_by(FileRecord.category)
                   .all()
               )
               return {
                   row.category or "Other": {
                       "count": row.count,
                       "total_size_bytes": row.total_size or 0
                   }
                   for row in results
               }
           finally:
               db.close()
       ```
